# 🚀 缘起：填充灵魂的赛博旅程

::: tip 碎碎念
写下这段话时，我正坐在电脑前盯着模型的训练进度。在那些大把的闲暇时刻，我总觉得要做些事情来填充自己的灵魂。
:::

## 🛡️ 我的初衷：看不惯，所以站出来

> “我看不惯现在国内编程环境的混乱，我更看不惯营销号一点一点把迷茫的编程小白们的热情消耗殆尽。”

构思这个网站内容很久了，直到今日才决定开始做，因为我觉得**时间不等人**。

从接触第一个编程语言 **Python** 到现在开发**前后端分离**，植入 **Coze 智能体**，再到**训练并且改进模型**，我差不多用了 **5 年**。

## 🧗 我的“八十一难”

这一路的感受令我痛苦，但我应该庆幸一点：
1. **热爱为帆**：我大学学习的是我喜欢的专业。
2. **硬气为骨**：我很自信，甚至可以说是**自负**。
3. **勤奋为径**：我肯下功夫。

这一切的积累，都是为了有能力帮助天底下的编程小白开启他们的学习之路。

::: warning 郑重承诺
这里没有营销号的焦虑，只有最真实的经验。
:::
