# 🚀 2025：我的全栈进化史

::: tip ⚡ 速度与激情
**历时**：5 个月（7月 - 12月）  
**跨度**：从零基础前端 -> C语言底层 -> Java 全栈 -> 独立系统开发  
**状态**：每天学到凌晨 12 点的“疯狂模式”
:::

## ☀️ 盛夏：前端启蒙与后端初探

这一阶段，我快速建立了对 Web 开发的感性认识，并不满足于前端，迅速向后端延伸。

- **07月23日**：✅ **前端基础通关** (HTML, CSS, JavaScript) —— 网页终于能动了。
  
- **08月07日**：✅ **前端框架 Vue** 不再手写 DOM，体会到了数据驱动视图的快乐。

- **08月19日**：✅ **后端基础 Node.js** 第一次用 JavaScript 触碰服务器的世界。

---

## 🍂 金秋：语言底层与 Java 帝国

这两个月是最痛苦也是最充实的。我不仅补了计算机的底（C语言、网络），还啃下了庞大的 Java 体系。

- **09月15日**：💾 **MySQL 数据库** 学会了数据的持久化存储，不再只是玩“内存数据”。

- **09月21日**：🧠 **C 语言** 探究指针与内存，理解计算机的“Hello World”到底发生了什么。

- **09月27日**：☕ **Java 基础** 正式踏入 Java 的大门。

- **10月10日**：🔧 **Java 进阶** 面向对象、集合、IO流，地基打牢。

- **10月18日**：☕ **Java SE 完结** 标准版通关。

### 📅 10月疯狂冲刺周
> 这一周，我以天为单位吞噬知识。

- **10月19日**：🌐 **Java EE & 计算机网络**
- **10月20日**：📊 **Java EE 数据结构**
- **10月21日**：🔌 **JDBC** (Java 数据库连接)
- **10月26日**：⚡ **MyBatis** (ORM 框架完善)
- **10月28日**：🎯 **攻克 JUE** (阶段性技术节点)

---

## ❄️ 凛冬：框架整合与系统落地

基础打完，开始实战。从 Demo 到真正的业务系统，这是质的飞跃。

- **11月11日**：🏆 **首个项目落地** 光棍节的礼物——成功做出第一个完整项目！

- **11月22日**：🍃 **Spring Boot** **里程碑**：至此，Java EE 技术栈拼图完成。告别繁琐配置，拥抱约定大于配置。

- **12月08日**：🏗️ **通用脚手架完成** 为了后续开发，封装了自己的开发底座。

- **12月24日**：🏥 **心理健康预约系统脚手架** **平安夜的献礼**：结合医疗领域，完成了系统的核心骨架搭建。

---

::: info 🎓 总结
这一路走来，每一天都是在挑战认知的极限。未来的路（模型训练、智能体结合）只会更精彩。
:::
